# 📊 Financial Sales Analysis Dashboard

## Overview

This Power BI dashboard provides a comprehensive analysis of financial sales performance, enabling stakeholders to monitor key business metrics, identify profitable products, evaluate market performance across countries, and track sales trends over time.

The dashboard transforms raw financial data into actionable insights through interactive visualizations and dynamic filtering capabilities.

---

## Key Features

### 📈 KPI Monitoring
- Total Sales Overview
- Total Profit Overview

### 🌍 Regional Analysis
- Sales distribution across countries
- Country-level performance comparison

### 📦 Product Performance
- Profit contribution by product
- Identification of high-performing products

### 👥 Customer Segment Analysis
- Sales breakdown by business segment
- Segment-wise performance evaluation

### 📅 Trend Analysis
- Monthly sales trend visualization
- Time-based performance tracking

### 🔍 Interactive Filters
- Country Slicer
- Product Slicer

---

## Dashboard Visuals

| Visualization | Purpose |
